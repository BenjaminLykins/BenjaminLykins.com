Code Coverage Undergraduate Research Project · November 2014

As part of my undergraduate research, I created a Python script that would take a file (termed the target), compare it to a folder containing other files to compare it to (termed the corpus folder) and create a basic HTML file that would contain the exact text of the target file with the sequences that had been repeated in the corpus highlighted.

The primary purpose was researching how much of a program could be created by copying long sequences of words (termed tokens) in other programs. If we gain a better understanding of this, it may be possible to create programming languages that reduce redundancy of tokens between similar programs. A secondary purpose could comparing students' code to each other to see if they may have copied code from another student in a class.
